# Geetesh Portfolio

Premium personal portfolio for a Senior UI Developer (14+ years), built with React + TypeScript + Tailwind CSS + Vite.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- React Icons
- React Hook Form + Zod
- ESLint + Prettier

## Features

- Atomic Design component architecture (atoms, molecules, organisms, templates)
- Sticky navbar
- Hero with CTA buttons
- Animated skill cards
- Experience timeline
- Featured projects with hover effects and modal quick view
- Services and testimonials
- Resume download section
- Contact form with validation and toast notifications
- Scroll progress bar
- Back to top button
- Light and dark theme support
- Lazy-loaded sections for performance
- SEO metadata via react-helmet-async

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Scripts

```bash
npm run dev        # Start dev server
npm run build      # Type-check and production build
npm run preview    # Preview production build
npm run lint       # Lint source code
npm run format     # Format project files
npm run typecheck  # TypeScript checks only
```

## Content Customization

Update portfolio content in `src/data/portfolioData.ts`.
